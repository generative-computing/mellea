Mellea is a library for writing generative programs. Generative programming replaces
flaky agents and brittle prompts with structured, maintainable, and efficient AI
workflows.

## Python Library

- **[mellea.backends](/api/mellea/backends/backend)** — Backend implementations.
- **[mellea.core](/api/mellea/core/backend)** — Core Library for Mellea Interfaces.
- **[mellea.formatters](/api/mellea/formatters/chat_formatter)** — Formatters.
- **[mellea.helpers](/api/mellea/helpers/async_helpers)** — Various Helpers and Utilities.
- **[mellea.plugins](/api/mellea/plugins/plugins)** — Mellea Plugin System — extension points for policy enforcement, observability, and customization.
- **[mellea.stdlib](/api/mellea/stdlib/context)** — The mellea standard library.
- **[mellea.telemetry](/api/mellea/telemetry/telemetry)** — OpenTelemetry instrumentation for Mellea.

## CLI (`m`)

- **[m alora](/api/cli/alora/commands)** — alora command group package for training and uploading adapters.
- **[m decompose](/api/cli/decompose/decompose)**
- **[m eval](/api/cli/eval/commands)** — CLI for test-based evaluation
- **[m m](/api/cli/m)** — Entrypoint for the M CLI.
